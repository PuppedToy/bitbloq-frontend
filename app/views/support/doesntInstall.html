¿Tiene un sistema operativo certificado?

  • S.O. certificados:

    • Windows 7
    • Windows 8.1
    • Windows 10
    • Ubuntu 14.04
    • Mac OSX 10.10+ (Solo 64b)
    • Chrome OS (chromebooks))

  • Otros S.O:

    • Windows XP
    • Debian
    • Mint
    • Xubuntu
    • OpenSuse

¿Su S.O es Vitalinux?:

  • Pueden solicitar al soporte oficial que lo instalen en todo el centro, de forma remota, a través de: Soporte Vitalinux
  • En el caso de querer instalarlo a mano:
    • sudo apt-get update
    • sudo apt-get install vx-dga-l-web2board-bq
    • Recuerde reiniciar el ordenador

¿Su S.O es MAX9?:

  • Si tiene dificultades, instale la última versión del standart_web2board_installer incluido en el paquete correspondiente a la arquitectura de su sistema: web2board para Linux 32 o web2board para Linux 64.
